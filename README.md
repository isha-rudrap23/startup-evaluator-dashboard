# Startup Evaluator Dashboard 🚀

A web application that evaluates startup ideas using a scoring algorithm and displays analytics through a modern dashboard.

---

## 📌 Features

- Startup idea evaluation system
- Automated scoring algorithm
- Dashboard analytics
- Top startup leaderboard
- Search and filtering
- CRUD operations (Create, Read, Update, Delete)
- Score visualization charts

---

## 🛠 Tech Stack

Backend  
- Java
- Spring Boot

Database  
- MySQL

Frontend  
- HTML
- CSS
- JavaScript

Libraries  
- Chart.js (analytics charts)
- Font Awesome (icons)

---

## ⚙️ How It Works

Users enter a startup idea with the following parameters:

- Market Size
- Competition
- Cost
- Scalability
- Innovation

The system calculates a **startup viability score** and stores the idea in the database.

The dashboard then shows:

- KPI statistics
- Startup leaderboard
- Score analytics charts
- Search and filtering options
